    • Media Queries

    Media queries allow responsive design based on screen size.

    You can change layout, font size, or visibility based on conditions.

    @media (max-width: 768px) { ... }
  • max-width
  • min-width
    • Responsive Images

    Images can resize automatically with the container.

    Use srcset and picture for different resolutions.

    <img srcset="small.jpg 480w, large.jpg 1024w">
  • Optimized for mobile
    • Mobile First Design

    Design for mobile first, then progressively enhance for larger screens.

    Improves performance and prioritizes essential content.

    @media (min-width: 768px) { ... }
